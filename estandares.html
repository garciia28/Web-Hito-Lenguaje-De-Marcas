ESTÁNDARES

Los estándares web se introdujeron para proteger el ecosistema web, para mantenerlo abierto, gratuito y accesible para todos.

Cuando se introdujeron los estándares, se incitó a los navegadores a adherirse a una forma estandarizada de hacer las cosas, esto resultó una compatibilidad cruzada cada vez más fácil para los creadores y no hubo necesidad de crear múltiples versiones para cada sitio web.

Tipos de estándares

  • HTML: define la estructura de los documentos.
  • CSS: sirve de base para bastantes tecnologías.
  • XML: permite asignar estilos para representar los documentos.
  • JavaScript: permite otorgar dinamismo y funcionalidad a la página web.

¿Cómo se crean los estándares web?

Para la creación de los estándares web se requiere de un proceso controlado y consta de varias etapas que aseguran la calidad de la especificación.

Este proceso permite intervenir a todos los usuarios de las tecnologías, cuyo objetivo es aportar su conocimiento y opiniones para la mejora de los documentos


ORGANISMOS

W3C

Es una comunidad internacional en la que las organizaciones trabajan conjuntamente para desarrollar estándares web.

Esta comunidad promueve el uso de estándares para reducir el coste y complejidad del desarrollo.
