CSS SCSS
Contiene todo tipo de funcionalidades para crear un buen CSS y crear una buena hoja de estilos y darle un buen estilo a tu pagina web. SCSS contiene todas las funciones de CSS y contiene más funciones que no están presentes en CSS, lo que lo convierte en una buena opción para que los desarrolladores lo usen.
El CSS ofrece ventajas como una mayor comodidad para crear hojas de estilo y algunas ayudas para crear las mismas. SCSS está lleno de características avanzadas.SCSS ofrece variables, puede acortar su código usando variables. Es una gran ventaja sobre el CSS convencional.
